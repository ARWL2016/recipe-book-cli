Angular 2 Recipe Book App

This application was built with Angular 2, Typescript, and Bootstrap to meet the criteria of the FreeCodeCamp challenge Build a Recipe Box. An alternative version of this app, built with React, can be seen here.

The data for this application is kept in the browser local storage, and will persist between sessions.

The form is implemented using the reactive form style.
