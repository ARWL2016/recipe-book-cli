Recipe Book

This application was built with Angular 4.0, Typescript 2.2, and Bootstrap 3.3.7 to meet the criteria of the FreeCodeCamp challenge Build a Recipe Box. An alternative version of this app, built with React, can be seen here.

Application features:

  • Application data cached in browser local storage
  • Form implemented using the Angular reactive form style
  • Notifications created with Toastr
  • Page animations require the @angular/animations module

The source code can be found on Github. For more projects, see my portfolio.
